### 🌟 Dự án tiêu biểu

#### 1. Modern E-commerce System (React 19 & FastAPI)
*Kiến trúc micro-framework hiện đại, tập trung vào tốc độ và trải nghiệm người dùng.*
- **Backend:** Xây dựng RESTful API bất đồng bộ (Async) bằng **FastAPI**, quản lý DB qua **SQLAlchemy 2.0** và migration bằng **Alembic**.
- **Frontend:** SPA hiệu suất cao với **React 19** và **Vite 8**, styling bằng **Tailwind CSS v4**.
- **AI Feature:** Tích hợp **Google Gemini API** làm chatbot hỗ trợ khách hàng thông minh.

#### 2. Native Full-stack Web (Core PHP & Vanilla JS)
*Dự án nền tảng được xây dựng 100% từ "Zero", không phụ thuộc framework.*
- **Core Logic:** Tự định nghĩa cấu trúc API, xử lý kết nối MySQL qua `mysqli`.
- **Custom Lib:** Tự phát triển thư viện `SimpleSMTP.php` để xử lý mail và tích hợp thanh toán qua **VietQR**.
- **Ý nghĩa:** Đây là dự án giúp mình làm chủ hoàn toàn luồng dữ liệu Client-Server trước khi chuyển sang các framework hiện đại.
